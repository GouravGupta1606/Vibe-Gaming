🧾 About Vibe Gaming

Welcome to Vibe Gaming — a personal web-based gaming platform developed by Gourav Gupta. This project showcases my skills in front-end web development and game programming using HTML, CSS, and JavaScript.

"No downloads. No installations. Just click, play, and enjoy."

Vibe Gaming features a collection of browser-based games designed to provide a smooth and engaging experience across devices. The project focuses on creating lightweight, accessible, and entertaining games that run directly in the browser.

From classic games like Tic-Tac-Toe and Memory Match to interactive experiences such as Endless Runner and Brick Breaker, every game has been designed, developed, tested, and refined independently as part of my learning journey.

Through this project, I have strengthened my understanding of responsive design, JavaScript game logic, DOM manipulation, animations, UI/UX design, and problem-solving while building real-world applications.

Why "Vibe Gaming"? Because gaming should be simple, fun, and accessible. The goal is to combine modern web technologies with the charm of classic browser games while continuously learning and improving as a developer.

This project serves as both a learning experience and a portfolio showcase, demonstrating my passion for web development and interactive user experiences.
